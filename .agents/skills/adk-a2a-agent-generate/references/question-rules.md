# Question Rules

Ask only when the missing information blocks a safe boundary decision or runnable implementation.

Ask for:

- missing stable agent identity
- ambiguous owner or domain boundary when it affects A2A
- target language or runtime when files must be generated
- unavailable source files explicitly referenced by the user

Do not ask for:

- prior deltas on a cold-start run
- complete input/output contracts when a sparse first pass is acceptable
- A2A decisions that the skill should make from evidence
